
The bitwise operators expect integer input (and will truncate float values).

| operator           | description            |
|--------------------|------------------------|
| [\&\&](../andand)  | logical `and`          |
| [\|\|](../oror)    | logical `or`           |
| [\&](../and)       | bitwise `and`          |
| [\|](../or)        | bitwise `or`           |
| [\>\>](../rshift)  | right signed bit shift |
| [\<\<](../lfshift) | left signed bit shift  |
